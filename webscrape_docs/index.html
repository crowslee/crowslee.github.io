webscrape adventure

little program that puts the user in the perspective of a webscraper trawling a webpage using the language of a first person text adventure, presented via video documentation showing examples of how different sites are displayed and navigated using the interface. built using node.js, express, p5 and RiTa.

by mateo reggie for computational poetics "dice in the browser", fall 2025
